# AlarmClock
WnStock是个开源的闹钟软件，使用QT5.9+vs2017开发。

### 功能列表如下：

1.可同时设置两组闹铃。

2.可选择闹铃音乐。

3.灵活设置闹铃提示语。

4.可最小化到托盘。

5.整点语音报时。
